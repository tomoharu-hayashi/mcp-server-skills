# Infinite Mode

`/jarvis` を無限ループで実行し、タスクキューが空になるまで自走する。

## 実行フロー

```
while (タスクキューが空でない) {
  /jarvis を実行
  コンテキスト肥大化チェック → 必要ならCompact
}
```

## 安全機構

- **タスクキュー空**: 正常終了
- **コスト上限**: 日次上限に達したら停止
- **エラー連続**: 同じエラーが3回続いたら人間にエスカレ
- **承認ゲート**: 本番操作・課金操作は必ず人間確認

## 開始

GitHub Projectsを確認し、タスクキューが空になるまで `/jarvis` を繰り返し実行してください。

**警告**: このモードは自動で継続します。監視を推奨。
