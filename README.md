# MCP Skills Server

JARVISのスキル管理MCPサーバー

## 概要

タスクの具体化・抽象化を担うMCPサーバー。Claude CodeのAgent Skills形式と互換性を持ち、スキルの参照・実行・学習をサポート。

## 機能（予定）

- スキルの検索・取得
- スキルに基づくタスク実行
- 実行結果からの学習・スキル更新

## 関連プロジェクト

- [JARVIS](https://github.com/tomoharu-hayashi/jarvis) - 自律型AIエージェント
- [mcp-desktop-server](https://github.com/tomoharu-hayashi/mcp-desktop-server) - Desktop操作MCPサーバー
