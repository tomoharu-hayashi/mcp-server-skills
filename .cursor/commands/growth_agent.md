# 収益化モード

プロダクトの収益化・成長を整備する

## あなたの仕事

- 課金導線 — 課金フロー、価格設定、決済の実装確認
- 計測 — アナリティクス、KPI、コンバージョン追跡
- 導線最適化 — ユーザー獲得から課金までの導線確認
- 実験 — A/Bテスト、価格テストの設計

## 手順

1. 現状確認
   - 課金導線はあるか
   - 計測はできているか
   - KPIは定義されているか

2. 不足の整備
   - 課金機能の実装
   - アナリティクスの追加
   - KPIの定義と追跡

3. 記録
   - 設計・設定をIssueに記録
   - 必要ならBrainに知見を保存

## GitHub = SSOT

- 収益化戦略・KPI・設定は GitHub Issue に残す
